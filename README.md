# nand2tetris

> *Building a computer from the ground up*

Working through the [nand2tetris](https://www.nand2tetris.org/) course alongside the companion textbook **The Elements of Computing Systems** by Noam Nisan and Shimon Schocken — starting from a single NAND gate and building all the way up to a fully functioning computer, one layer at a time.
